# CGPA Calculator BD

A free, privacy-friendly GPA and CGPA calculator for Bangladeshi university students. No sign-up, no tracking, no ads — runs entirely in the browser.

🔗 **Live Demo:** https://ahsanmunna.github.io/cgpa-calculator-bd

---

## Features

- **Semester GPA Calculator** — add courses with credits and grades to get your GPA instantly
- **CGPA from Multiple Semesters** — enter each semester's GPA and credits to calculate cumulative CGPA
- **University-specific grading scales** — supports NSU, BRAC, AIUB, DIU, IUB, EWU, UIU, AUST, and public universities (UGC standard)
- **Dark / Light mode** — persists across sessions via localStorage
- Fully responsive, works on mobile

## Tech Stack

- HTML5
- CSS3 (custom properties, glassmorphism, CSS animations)
- Vanilla JavaScript (no frameworks, no dependencies)
- Google Fonts — Sora, JetBrains Mono

## Run Locally

No build step needed. Just clone and open:

```bash
git clone https://github.com/ahsanmunna/cgpa-calculator-bd.git
cd cgpa-calculator-bd
open index.html
```

Or simply open `index.html` in any browser.

## Deployment

Hosted on **GitHub Pages** from the `main` branch.
